Tentang Saya

Mahasiswa Teknik Informatika

Saya adalah mahasiswa semester 3 Program Studi Teknik Informatika. Saya memiliki minat yang besar dalam pengembangan web, khususnya di bidang front-end development dan user experience design.

Selama perkuliahan, saya telah mempelajari berbagai teknologi web seperti HTML, CSS, JavaScript, dan framework modern seperti React dan Vue.js. Saya juga memiliki pengalaman dalam pembuatan aplikasi web yang responsif, interaktif, dan user-friendly.

Selain itu, saya aktif dalam organisasi kampus dan telah mengikuti berbagai kompetisi programming serta hackathon. Saya selalu bersemangat untuk belajar teknologi baru dan menerapkannya dalam proyek-proyek nyata yang memberikan dampak positif.
